Одностраничники

сайты которые представляют вашу компанию в
виде вашей личной визитки - с гиперссылками на ваши
внешние ресурсы/ социальные сети. Один из самых
бюджетных вариантов сайта, но не менее привлекательный
вариант, использование:

  • HTML (Перенос макета в код)

  • SCSS (Возможность использования препроцессора)
  • CSS Animation (Возможность использования анимации)
